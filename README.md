Around 2016 I started a security project for analyzing Ethereum bytecode in such a way that branches in execution could be simulated simultaneously without introducing a large time overhead. This evolved to being able to simulate the Ethereum virtual machine & other Ethereum-compatible smart contracts. As of October 2020 I'm working fulltime on these projects through my company Minesec. We currently do a lot of Kotlin & a bit of mathematics to create new tools and systems supporting the automated analysis of the entire Ethereum blockchain. We prove & disprove security aspects for deployed smart contracts & potentially interact with them depending on the outcome of our analysis. If you'd like for your smart contract to be analyzed by our product, simply deploy it to the Ethereum main net and allocate it some Ether. We'll see if we can pick it up. :^)
